# Тест-кейсы: Авторизация

## TC-AUTH-001: Успешная авторизация (Позитивный)

**Приоритет:** Критический  
**Предусловия:** 
- Пользователь зарегистрирован
- Username: `ivan_petrov`
- Пароль: `Client123`

### Шаги:
1. Перейти на страницу входа `/accounts/login/`
2. Ввести username: `ivan_petrov`
3. Ввести пароль: `Client123`
4. Нажать "Войти"

### Ожидаемый результат:
- ✅ Пользователь авторизован
- ✅ Перенаправление на главную страницу
- ✅ Отображается сообщение "Добро пожаловать, Иван Петров!"
- ✅ В навигации отображается имя пользователя
- ✅ Доступны ссылки "Мои заказы" и "Профиль"

### Фактический результат:
**Статус:** ✅ PASSED
