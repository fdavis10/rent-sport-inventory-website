## 1. НАЗНАЧЕНИЕ ПРОГРАММЫ

### 1.1 Общее описание

Веб-приложение "Система аренды спортивного инвентаря" предназначено для автоматизации процесса аренды спортивного оборудования (лыжи, сноуборды, коньки и аксессуары).

### 1.2 Целевая аудитория

- **Клиенты** - пользователи, арендующие инвентарь
- **Менеджеры** - сотрудники, управляющие заказами
- **Администраторы** - персонал с полным доступом к системе

### 1.3 Основные функции (Версия 1.0)

1. Просмотр каталога спортивного инвентаря
2. Регистрация и авторизация пользователей
3. Управление профилем пользователя
4. Добавление товаров в корзину
5. Базовый расчет стоимости аренды
6. Оформление заказов
7. Просмотр истории заказов
8. Управление инвентарем (для администраторов)

## 2. ТРЕБОВАНИЯ К ФУНКЦИОНАЛЬНЫМ ХАРАКТЕРИСТИКАМ

### 2.1 Регистрация и авторизация

#### 2.1.1 Регистрация пользователя

**Входные данные:**
- Имя пользователя (username) - уникальное, обязательное
- Email - уникальный, обязательный
- Имя - обязательное
- Фамилия - обязательное
- Телефон - опциональное
- Пароль - обязательное
- Подтверждение пароля - обязательное

**Требования к паролю:**
- Длина: от 4 до 16 символов
- Обязательно наличие заглавных букв
- Обязательно наличие цифр
- Запрещены символы: * & { } | +

**Выходные данные:**
- Создание учетной записи в базе данных
- Автоматическая авторизация
- Присвоение роли CLIENT по умолчанию

#### 2.1.2 Авторизация

**Входные данные:**
- Имя пользователя или email
- Пароль

**Выходные данные:**
- Создание пользовательской сессии
- Перенаправление на главную страницу или в админ-панель (для менеджеров/админов)

### 2.2 Управление каталогом

#### 2.2.1 Просмотр каталога

**Функционал:**
- Отображение всех активных товаров
- Карточка товара содержит:
  - Изображение
  - Название
  - Категория
  - Размер
  - Состояние
  - Цена за день
  - Статус доступности

#### 2.2.2 Поиск

**Входные данные:**
- Текстовый запрос

**Логика:**
- Поиск по названию, описанию, бренду

#### 2.2.3 Фильтрация

**Параметры:**
- По категории
- По диапазону цен (мин-макс)
- По состоянию (отличное, хорошее, удовлетворительное)
- По доступности (в наличии / нет в наличии)

#### 2.2.4 Сортировка

**Варианты:**
- По дате добавления (новые/старые)
- По цене (возрастание/убывание)
- По названию (А-Я / Я-А)

### 2.3 Корзина и оформление заказа

#### 2.3.1 Добавление в корзину

**Входные данные:**
- ID товара
- Количество
- Дата начала аренды
- Дата окончания аренды

**Валидация:**
- Количество не превышает доступное
- Дата начала не в прошлом
- Дата окончания позже даты начала

#### 2.3.2 Расчет стоимости

**Формула:**

Количество дней = (Дата окончания - Дата начала) + 1
Базовая стоимость = Цена за день × Количество × Дни
Итоговая стоимость = Базовая стоимость

#### 2.3.3 Оформление заказа

**Процесс:**
1. Проверка заполненности корзины
2. Отображение итоговой информации
3. Возможность добавить комментарий
4. Создание заказа со статусом PENDING
5. Перенос позиций из корзины в заказ
6. Уменьшение доступного количества товара
7. Очистка корзины

### 2.4 Управление заказами

#### 2.4.1 Статусы заказов

- **PENDING** - Ожидает подтверждения
- **CONFIRMED** - Подтвержден
- **ACTIVE** - Активная аренда
- **COMPLETED** - Завершен
- **CANCELLED** - Отменен

#### 2.4.2 Просмотр заказов

**Для клиента:**
- Список своих заказов
- Фильтрация по статусу
- Детальная информация о каждом заказе

**Для менеджера:**
- Список всех заказов
- Возможность изменения статуса
- Подтверждение заказов
